# cthrice

This is the C transpiler for the Thrice language, written in C.

## Naming Convention

```C
/*
 * CTHRICE_CONSTANT
 * CTHRICE_MACRO_CONSTANT
 * CTHRICE_ENUM_CONSTANT
 * Cthrice_Type
 * cthrice_Global
 * cthrice_function
 * cthrice_macro_function
 * mbrvar
 * lclvar
 */
```
